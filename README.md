# Data Cleaning & Exploratory Data Analysis — SQL

## Objective
Cleaning a raw dataset and performing exploratory data analysis (EDA) in SQL 
to identify patterns, trends, and data quality issues ahead of deeper analysis.

## Approach
- **Data Cleaning**: identified and handled duplicates, standardized inconsistent 
  formatting, and addressed null/blank values
- **Exploratory Data Analysis**: used aggregate functions, grouping, and filtering 
  to surface trends and summary statistics in the cleaned dataset

## Skills Demonstrated
- Data cleaning (deduplication, standardization, null handling)
- Aggregate functions (SUM, COUNT, AVG, MIN, MAX)
- GROUP BY / filtering for exploratory analysis
- Query structuring for readability
